warning lf will be replaced by crlf
  git init
  git add . 出现提示：
  windows中的换行符为 CRLF， 而在linux下的换行符为：LF
  git init，文件中的换行符为LF， 当执行git add .时，系统提示：LF 将被转换成 CRLF
解决办法：
  rm -rf .git //移除.git文件
  git config --gobal core.autocrlf false  //禁用自动转换
